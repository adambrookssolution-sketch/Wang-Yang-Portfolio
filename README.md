# Wang Yang (王洋) — Portfolio

Personal portfolio website for **Wang Yang**, a Full Stack Engineer based in Wuhan, Hubei, China.

Built with **React 19 + Vite**, animated with **Framer Motion**, and styled from scratch.

## Stack

- React 19, Vite 8
- Framer Motion, Lenis
- React Icons

## Scripts

```bash
npm install
npm run dev      # start dev server
npm run build    # production build
npm run preview  # preview production build
```
